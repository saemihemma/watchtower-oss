# No Skills (Baseline)

This directory is intentionally empty of skills. It represents the bare LLM with no skill library loaded.

## Purpose

Watchtower compares Side A (LLM + skills) against Side B (LLM + no skills). This directory is Side B. The only variable between the two sides is whether skills are present in context.

## Why This File Exists

Watchtower requires at least one SKILL.md to validate a directory as a skill library root. This file satisfies that requirement while providing zero actual skill content. The LLM receives no routing, no process discipline, no structured methodology — just its own training.
